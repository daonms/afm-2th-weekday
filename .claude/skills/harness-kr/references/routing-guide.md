# 한국어 작업 지시 라우팅 가이드

이 파일은 사용자의 한국어 자연어 지시를 받아 적합한 에이전트로 라우팅하는 기준입니다.
Claude는 이 가이드를 참고하여 에이전트를 선택합니다.

## 불명확한 지시 처리

사용자의 지시가 여러 에이전트에 해당될 수 있을 때:

```
어떤 작업을 원하시나요?

1. 프론트엔드 UI 개발 → 프론트엔드-개발자
2. 백엔드 API 개발 → 백엔드-개발자
3. AI 기능 추가 → AI-개발자

번호를 선택하거나 더 자세히 말씀해 주세요.
```

---

## 복합 작업 처리

하나의 지시가 여러 에이전트를 필요로 할 때 순서대로 진행:

**풀스택 앱 개발**:
1. `기획자` → 기획·설계
2. `백엔드-개발자` → API 서버
3. `프론트엔드-개발자` → UI
4. `검토자` → 코드 리뷰
5. `배포자` → 배포

**AI 기능 있는 웹앱**:
1. `AI-개발자` → Claude API 연동 로직
2. `프론트엔드-개발자` → UI 구현
3. `백엔드-개발자` → 서버 연동 (필요시)
