노트북 화면.

창백한 빛이 한 사람의 얼굴을 비춘다. 눈 밑에는 옅은 그늘이 졌지만, 눈동자는 — 살아있다. 화면 위에서 무언가를 찾듯, 줄줄이 늘어선 코드들을 따라 시선이 움직인다.

터미널 창이 깜박인다.

```
$ python main.py
Traceback (most recent call last):
  File "main.py", line 23, in <module>
AttributeError: 'NoneType' object has no attribute 'split'
```

*에러.*

*또.*

한숨이 나오지 않는다. 오히려 — 입꼬리가 미세하게 올라간다. 열두 시간째다. 이쯤 되면 에러 메시지도 오래된 친구처럼 느껴진다.

*나는 오늘도 이 방에서, 세상과 싸우고 있다.*

## EPISODE 3. 열두 시간의 해부학

**INT. 작은 방 — 하루의 타임랩스**

카메라가 방의 한 구석에 고정된다. 시간이 빠르게 흐른다.

**오전 9시.** 커피 한 잔. 노트북이 켜진다. 강의 화면이 열린다. 강사의 목소리가 방 안을 채운다. 손이 키보드 위에 올려진다. 조심스럽게, 처음 악기를 잡는 사람처럼.

**오후 1시.** 점심은 대충이다. 컵라면 물을 붓는 4분 동안에도 눈은 화면을 떠나지 않는다. 뭔가 이해될 것 같은 느낌. 잡힐 듯 잡히지 않는 그 감각.

**오후 4시.** 에러. 에러. 또 에러. 스택 오버플로우를 뒤지고, 공식 문서를 읽고, AI에게 물어보고, 다시 코드를 고친다. 이 사이클이 한 시간에 열두 번쯤 반복된다.

**오후 7시.** 뭔가 — 된다.

**클로즈업 — 손가락이 엔터 키를 누른다**

```
$ python main.py
모델 로딩 완료...
✓ 정확도: 87.3%
```

숨을 참았다는 걸, 이때서야 알았다. 후우— 하고 내쉬는 숨.

그리고 — 눈물이 났다.

등받이에 몸을 기댄 채, 천장을 올려다본다. 천장이 흐릿하게 번진다. 손등으로 눈가를 훔친다. 한 번. 두 번. 그래도 멈추지 않는다.

*왜 우는 거지. 고작 코드 하나 돌아간 건데.*

아니다. 고작이 아니다. 세 시간을 붙잡았다. 스택 오버플로우를 스물네 번 새로고침했다. 변수명을 열일곱 번 바꿨다. 포기하고 싶었던 순간이 적어도 다섯 번은 있었다. 그 모든 시간이, 이 터미널 한 줄에 담겨 있다.

*이 87.3%가 뭐라고. 이게 뭐라고 이렇게 눈물이 나는 거지.*

노트북 화면 위로 눈물 한 방울이 떨어질 뻔한다. 황급히 고개를 돌린다. 아무도 없는 방인데, 왜 부끄러운 걸까. 아니 — 부끄러운 게 아니다. 이건 **자랑스러운 거다.**

> *(BGM: 피아노 선율에 첼로가 조심스럽게 합류한다. 선율이 떨린다 — 마치 눈물처럼.)*

**오후 9시.** 다시 새로운 챕터를 연다. 기쁨은 연료다. 아직 배가 부르다.

**밤 11시.** 눈이 뻑뻑하다. 손목이 묵직하다. 그러나 멈추지 않는다. 왜냐면 —

*오늘 처음으로, 내가 만든 코드가 살아 숨쉬는 걸 봤으니까.*

## EPISODE 4. 새벽의 각성 — 나는 왜 여기 있는가

**INT. 작은 방 — 새벽 1시**

코드를 보다가, 문득 손이 멈춘다.

화면 속 수십 줄의 코드. 일주일 전만 해도 외계어처럼 보이던 것들이, 지금은 — 어렴풋이 읽힌다. 완벽히 이해한 건 아니다. 그러나 분명히, **어제의 나보다는 조금 더 보인다.**

나레이션이 흐른다.

*나는 지금 초보다.*

*이 사실을 숨기지 않겠다. 오늘도 기초 문법에서 막혔고, 들여쓰기 하나 때문에 삼십 분을 날렸고, 강사가 "쉽다"고 한 개념을 나는 다섯 번 돌려봤다.*

*그래도.*

카메라가 천천히 줌인된다. 노트북 화면에는 오늘 처음으로 완성한 작은 프로그램이 실행되고 있다. 보잘것없다. 실용적이지도 않다. 그러나 이것은 — **내가 만든 것이다.**

*누군가는 AI가 세상을 바꿀 거라고 말한다. 나는 그 말을 믿는다. 그리고 그 누군가 중에, 내가 있어야겠다고 — 오늘 밤 처음으로, 조심스럽게 생각했다.*

> *"선구자란, 처음부터 앞서간 사람이 아니라 — 남들이 멈출 때도 걸어간 사람이다."*

## EPISODE 5. 에러 로그 — 오늘의 성적표

**INT. 작은 방 — 새벽 2시**

하루를 정리한다. 노트에 오늘의 에러들을 적는다.

```
오늘의 에러 목록:
1. AttributeError × 7회
2. IndentationError × 12회
3. KeyError × 3회
4. 그냥 멍하니 화면 보기 × 무수히
```

펜이 잠시 멈춘다. 그리고 그 아래에 천천히 덧붙인다.

```
오늘의 수확:
1. 처음으로 API 연결에 성공함
2. 에러 메시지를 읽는 법을 배움
3. 모르는 걸 물어보는 것이
   부끄러운 일이 아님을 배움
4. 열두 시간이 세 시간처럼 느껴졌음
```

*에러 로그가 많을수록, 배운 것도 많다는 걸 — 이제는 안다.*

> *(BGM: 선율이 조금 더 따뜻해진다)*

창문 밖으로 3월의 밤이 깊어간다. 어딘가에서 harbor.school의 같은 기수들도 지금쯤 이 새벽과 싸우고 있을 것이다. 각자의 방에서, 각자의 에러 앞에서.

*우리는 혼자지만, 혼자가 아니다.*
