# MyAide Agent Studio — Mission

## 목표

수강생이 `에이전트`와 `스킬`의 역할을 이해하고, 하나의 작업 요청을 어떤 실행 순서로 나눠야 하는지 연습한다.

## 문제

AI 도구를 처음 쓰는 사람은 보통 이렇게 막힌다.

- 어떤 에이전트에게 맡겨야 하는지 모른다.
- 스킬이 왜 필요한지 모른다.
- 작업 지시가 너무 두루뭉술해서 결과가 흔들린다.
- 검토 없이 바로 구현해서 안전·품질 문제가 생긴다.

## 해결

MyAide Agent Studio는 사용자의 한국어 요청을 입력받아 다음을 보여준다.

1. 어떤 에이전트가 필요한가
2. 어떤 스킬을 쓰면 좋은가
3. 어떤 순서로 작업해야 하는가
4. 검토자는 무엇을 확인해야 하는가
5. 성향분석과 회의 기록을 어떻게 전략 가설로 바꿀 수 있는가

## 수업 범위

- React CDN
- Tailwind CDN
- localStorage
- 단일 HTML 앱
- 한국어 에이전트 라우팅
- 선천/후천 성향분석 실습
- Notion AI 회의 녹음 결과 전략화 실습

## 비범위

- 실제 AI API 호출
- 실제 메시지 전송
- 실제 녹음 파일 업로드/저장
- 서버/DB 저장
- 운영 루트 기능 변경
